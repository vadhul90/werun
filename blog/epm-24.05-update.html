Survey Note: Detailed Analysis of Sales Planning Cloud and ePBCS Updates for May 2024

This note reviews the May 2024 updates (24.05) for Oracle’s Sales Planning Cloud and ePBCS, based on documentation as of March 10, 2025, focusing on data imports and platform compatibility.

Background and Context

May enhances data handling and cross-platform support in Oracle EPM Cloud.

Sales Planning Cloud Updates: Detailed Breakdown

  • Pipeline Feature for Batch Imports:
    • New pipeline simplifies batch data imports.
    • Impact: Streamlines data loading processes.
    • Implementation Consideration: Replace old batch tools with pipeline.
    • Business Benefit: Faster data updates, e.g., importing sales data in bulk.

ePBCS Updates: Detailed Breakdown

  • Smart View Extension for Mac:
    • Enhances Smart View performance on macOS.
    • Impact: Improves usability for Mac users.
    • Implementation Consideration: Update Mac clients to latest version.
    • Business Benefit: Consistent experience across platforms.
