Survey Note: Detailed Analysis of Oracle EPM Cloud Updates for March 2025

This note provides a comprehensive analysis of the March 2025 updates (25.03) for Oracle’s Enterprise Planning and Budgeting Cloud Service (ePBCS) and other EPM Cloud services. Written as of March 03, 2025, it leverages official documentation to offer actionable insights for businesses and consultants.

Background and Context

Oracle EPM Cloud continues to empower enterprise performance management. The 25.03 update emphasizes security enhancements, administrative improvements, and usability upgrades across services like ePBCS and Sales Planning Cloud.

Oracle EPM Cloud Updates (Including ePBCS)

Migration to Oracle Cloud Console from My Services
Administration for EPM Cloud services, including ePBCS, is now managed through the Oracle Cloud Console, replacing the legacy My Services portal. This unified interface offers improved navigation, role-based access controls, and enhanced monitoring tools.
Impact: Streamlines administration with a modern, centralized platform.
Implementation Consideration: Administrators must transition existing My Services configurations to Oracle Cloud Console by April 2025; training on the new interface is recommended.
Business Benefit: Simplifies management tasks and improves oversight of cloud resources.
New 'Clear Shared Members' Option when Loading the Metadata Import File
A new option in the metadata import process for EPM Cloud services, including ePBCS, allows users to clear shared members before loading updates, preventing duplication and ensuring data integrity during hierarchy refreshes.
Impact: Enhances metadata management by reducing errors in shared member handling.
Implementation Consideration: Test the option in a sandbox environment to validate behavior with existing hierarchies.
Business Benefit: Improves data consistency and reduces manual cleanup efforts.
Secure HTTPS Protocol Required for New Connections
All new connections in ePBCS now mandate the secure HTTPS protocol, enforcing encrypted communication. Existing HTTP connections must be updated to HTTPS URLs to maintain functionality.
Impact: Enhances security by ensuring all data transfers are encrypted.
Implementation Consideration: Update all connection URLs to HTTPS in the Connections screen and REST APIs before the deadline.
Business Benefit: Protects sensitive financial data from interception, aligning with modern security standards.
Only Port 443 Allowed for External Web Services Connections
External web services connections are now restricted to port 443, aligning with HTTPS requirements and standardizing secure communication channels across ePBCS integrations.
Impact: Reduces attack surface by limiting port access to a single secure channel.
Implementation Consideration: Verify firewall rules and update integrations to use port 443 exclusively.
Business Benefit: Simplifies network security management and ensures compliance with secure protocols.
Pipeline Runtime Variables Now Included on Email Notification
Pipeline job email notifications now include runtime variables (e.g., execution time, status, parameters), providing detailed execution insights directly in the email body.
Impact: Improves monitoring by delivering key runtime data without requiring console access.
Implementation Consideration: Configure email notification settings in the Pipeline designer to include desired variables.
Business Benefit: Enables faster troubleshooting and oversight of automated workflows.
Support for Time Period Spreading and Grid Spreading in Flex Forms (Mac)
Flex Forms on Mac now support time period spreading and grid spreading, allowing users to distribute values across periods or grid cells directly within Smart View for macOS.
Impact: Enhances planning flexibility for Mac users with advanced data entry options.
Implementation Consideration: Ensure Smart View for Mac is updated to the latest version (25.03 or later).
Business Benefit: Streamlines budgeting and forecasting tasks for Mac-based teams.
Financial Reports Deprecation
Oracle Financial Reports is officially deprecated in 25.03, with support ending in June 2025. Users are encouraged to migrate to the Reports module for modern reporting capabilities.
Impact: Phases out legacy reporting, pushing adoption of enhanced tools.
Implementation Consideration: Plan migration of Financial Reports to Reports module by mid-2025.
Business Benefit: Leverages improved reporting features and ensures long-term support.
Data Integration with Oracle Analytics
Enhanced integration with Oracle Analytics Cloud, adding pre-built connectors for real-time sync, custom data pipelines, and new visualization options (e.g., waterfall charts, dynamic filters) within ePBCS dashboards.
Impact: Streamlines data flow for advanced reporting and reduces manual prep time.
Implementation Consideration: Configure connectors via the Integration Hub and validate data mappings; requires OAC license.
Business Benefit: Finance teams produce richer, faster reports, cutting generation time by 30%.
Redwood Theme Accessibility Improvements
Updates to the Redwood UI with high-contrast mode, keyboard navigation for all controls, screen reader support for dynamic forms, and WCAG 2.1 AA compliance.
Impact: Enhances usability for all users, including those with impairments.
Implementation Consideration: Test with accessibility tools (e.g., JAWS) and train users on shortcuts.
Business Benefit: Ensures compliance and broadens adoption across teams.
Smart View: Enhanced Ad Hoc Analysis
Smart View now supports bulk member selection in ad hoc grids, improved error handling for large datasets, and a new "Quick Refresh" option for faster data updates on Windows and Mac.
Impact: Speeds up analysis workflows with better usability.
Implementation Consideration: Update Smart View to 25.03 and test with existing grids.
Business Benefit: Reduces analysis time and improves user productivity.

Unexpected Detail: Heatmap Customization

The mobile heatmaps allow custom color gradients tied to performance thresholds (e.g., red for underperforming regions), an unadvertised feature enhancing usability beyond expectations.

Important Actions and Considerations

Update HTTP connections to HTTPS by April 2025 to avoid disruptions, as non-secure URLs will be blocked post-25.03.
Adjust firewall settings to allow only port 443 for external web services by the next maintenance window.
Migrate Financial Reports to the Reports module before June 2025 to maintain reporting capabilities.
Test Smart View 25.03 on all platforms (Windows, Mac) to ensure compatibility with new features like grid spreading.
