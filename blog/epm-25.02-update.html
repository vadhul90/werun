Key Planning Module Enhancements

Enhanced Sandbox Cloning

Previous Method


// 25.01 Sandbox Clone
Sandbox.clone("Q1_Forecast")
   .fromBase("Actuals")
   .withData()
   .execute()

Limitations:
- No partial data copy
- 30+ minute clone time for 5GB+ data

25.02 Improvements


// New 25.02 Options
SandboxOptions options = new SandboxOptions()
   .withData(filter: "Year = FY24")
   .withSecurity(roles: ["PLANNER"])
   .withNotification(email: "team@company.com")

Sandbox.clone("Q1_Scenario")
   .from("Q1_Actuals")
   .using(options)
   .asyncExecute()

58% faster

5GB sandbox clone

Storage Savings

40% reduction

25.02 Enhanced Methods

ValidationReport report = grid.validate()
   .checkFormulas()
   .checkCellText(MAX_LENGTH: 255)
   .checkCustomRules(ruleset: "FinancialPolicy")

if (report.hasErrors()) {
    emailService.sendAlert(report)
    ui.alert(report.summary)
}
